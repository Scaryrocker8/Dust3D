# Dust3D

![icon](Dust3D.svg)

A simple way to add dust to any 3d scene in Godot!

This was taken from perfoon's Abandoned Spaceship Godot Demo:

https://github.com/perfoon/Abandoned-Spaceship-Godot-Demo

I made the dust it's own scene that can be added to any other 3d scene!
